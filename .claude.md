# IrukaDark LP プロジェクトルール

## 多言語対応の重要ルール

このプロジェクトは日本語・英語の多言語対応を実装しています。

### HTML更新時の必須作業

**HTMLファイルのテキストを変更した場合は、必ず `app.js` の翻訳定義も更新すること。**

#### 更新手順

1. HTMLファイル内の `data-i18n` 属性を持つ要素を変更した場合
2. `app.js` の `translations` オブジェクトで対応するキーを検索
3. `translations.ja` と `translations.en` の**両方**を更新

#### 例

HTMLで変更:
```html
<h1 data-i18n="hero.title">新しいキャッチコピー</h1>
```

app.jsで更新:
```javascript
translations.ja['hero.title'] = '新しいキャッチコピー'
translations.en['hero.title'] = 'New Catchphrase'
```

### ファイル対応表

- `/index.html` → 英語版
- `/ja/index.html` → 日本語版
- `/app.js` → 両言語の翻訳定義を含む

### 翻訳キーの場所

`app.js` 内の `translations` オブジェクト:
- Line 2-146: 英語版 (`translations.en`)
- Line 147-290: 日本語版 (`translations.ja`)

## 作業時の注意点

- HTMLとJavaScriptの同期を忘れない
- 言語切り替えをテストして確認
- HTMLマークアップ（`<br>`, `<span>`等）も翻訳文字列に含める
