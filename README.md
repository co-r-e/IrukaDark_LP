# IrukaDark Landing Page

IrukaDark公式サイトのランディングページです。

## 開発サーバーの起動

```bash
npm run dev
```

開発サーバーが起動し、`http://localhost:3000`（または利用可能なポート）でアクセスできます。

## キャッシュバスターの更新

本番環境にデプロイする前に、キャッシュバスターを更新してください。これにより、ユーザーのブラウザが最新のCSS/JSファイルを確実に読み込むようになります。

### 自動更新（推奨）

```bash
npm run update-cache
```

このコマンドを実行すると：
- 現在の日時（YYYYMMDDHHMMSS形式）を自動取得
- すべてのHTMLファイル内のキャッシュバスターを一括更新
- 更新されたファイルのリストを表示

### 手動更新

スクリプトを使わずに手動で更新する場合：

1. 現在の日時を取得：
   ```bash
   date +"%Y%m%d%H%M%S"
   ```

2. すべてのHTMLファイルで `?v=旧日時` を `?v=新日時` に一括置換

## ファイル構造

```

│   ├── privacy.html

│   └── downloads/

├── app.js                  # メインJavaScript
├── components.js           # コンポーネント（ヘッダー/フッター）
├── dev-server.mjs          # 開発サーバー
└── update-cache-buster.mjs # キャッシュバスター更新スクリプト
```

## デプロイ前のチェックリスト

- [ ] キャッシュバスターを更新（`npm run update-cache`）
- [ ] すべてのページでスタイルが正しく表示されることを確認
- [ ] モバイル表示を確認（実機推奨）
- [ ] リンクが正しく動作することを確認

## 注意事項

- 開発サーバーは `Cache-Control: no-cache` を設定しているため、開発中はキャッシュバスターを更新する必要はありません
- 本番環境（GitHub Pages等）では、デプロイ前に必ずキャッシュバスターを更新してください
- キャッシュバスターは日時形式（`YYYYMMDDHHMMSS`）を使用しています
