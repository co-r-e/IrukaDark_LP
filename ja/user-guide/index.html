ユーザーガイド

IrukaDark をはじめて使うときから、ショートカット・スラッシュコマンド・通常チャットの活用まで、必要なポイントをまとめました。

初期設定

ショートカットやスクリーンショット解説をスムーズに使えるよう、最初に次の設定を済ませてください。

APIキー設定

  1. Google AI Studio で Gemini API キーを発行します（Vertex の鍵は非対応）。
  2. IrukaDark を開き、IrukaDark → AI Settings → Set GEMINI_API_KEY を選びます。
  3. 余分なスペースや引用符を付けずにキーを貼り付け、保存します。入力直後に検証が行われます。
  4. 環境変数で管理したい場合は、起動前に GEMINI_API_KEY（または GOOGLE_GENAI_API_KEY）を設定してください。

モデル設定

  1. 同じ AI Settings メニューで、まずは gemini-flash-lite-latest を選択するのがおすすめです（応答が最速）。
  2. より厚めの解説やウェブ検索が欲しいときは gemini-flash-latest に切り替えてください。
  3. モデルを固定したい場合は、環境変数 GEMINI_MODEL や WEB_SEARCH_MODEL を設定しておけます。
  4. 呼び出しが失敗した場合は、再度 lite 推奨モデルに戻すか、時間をおいてリトライすればエラー理由がチャットに表示されます。

アクセシビリティ許可

  1. 初めて Option+A を使うと、macOS から アクセシビリティ 許可のダイアログが表示されます。有効にすると選択テキストを読み取れます。
  2. Option+S で領域を切り取る際は 画面収録 の許可が必要です。「設定 → プライバシーとセキュリティ → 画面収録」で IrukaAutomation を許可してください。
  3. もしショートカットが効かなくなったら、アクセシビリティ／画面収録のリストで IrukaDark と IrukaAutomation が有効か確認します。
  4. 許可後は IrukaDark を再起動すると、ヘルパープロセスに設定が確実に反映されます。

Option+A — クイック解説

⌥+A

選択したテキストを短く要約します。ちょっとした疑問をすぐ確認したいときに便利です。

Option+Shift+A — 詳細解説

⌥+⇧+A

背景や追加情報を含めて丁寧に説明します。深い理解や手順が欲しいときに。

Option+Control+A — 図解

⌥+⌃+A

選択したテキストを図解画像にします。概念や関係性を視覚的に理解したいときに。

Option+Q — URLサマリー

⌥+Q

選択した URL を取得して整形し、重要度順に4文構成でサマリーを返します。

Option+Shift+Q — URLディープダイブ

⌥+⇧+Q

同じ URL 取得プロセスから、概要・重要ポイント・リスク・推奨アクションを含む構造化レポートを作成します。

Option+R — 翻訳

⌥+R

選択テキストを UI 言語へ自然に翻訳します。専門用語はなるべく残したままです。

Option+S — スクショ解説

⌥+S

画面の任意領域をドラッグで選択し、画像内テキストや文脈を抜き出して要約します。

Option+Shift+S — 詳細スクショ

⌥+⇧+S

選択領域の背景や意味、推奨アクションまで掘り下げた解説をリクエストします。

スラッシュコマンド

チャット入力欄で / を押すとコマンドパレットが開き、直前までの会話に対して各コマンドが動作します。

/clear

現在の会話履歴をリセットします。ウィンドウはそのままなのですぐに新しい質問を続けられます。

/compact

直近の履歴を短く圧縮し、次の回答がブレずに進むようトークンを節約します。

/next

前回の回答の続きを生成します。途中で切れたときや、さらに深掘りしたいときに。

/table

直前の出力を表形式に変換します。比較・整理が必要なときに便利です。

/what do you mean?

前の回答をより平易な言い回しで説明し直します。難しい内容の理解に。

/translate

サブメニューが開き、/translate_ja や /translate_fr など言語別コマンドで直前の返信を翻訳できます。

/web

/web on と /web off でウェブ検索モードを切り替え、/web status で状態を確認します（Google 検索対応モデルが必要）。

/contact

問い合わせページを開き、フィードバックや不具合報告をすぐに送信できます。

/slide

直前の会話内容からスライド画像を生成します。プレゼン資料や説明用ビジュアルをすぐ作りたいときに。

通常チャット

ショートカットやコマンドを使わなくても、常駐する IrukaDark ウィンドウで通常のチャットとして利用できます。

日常的な使い方

  • テキストボックスに入力し、Return で送信します。
  • 会話はリセットするまで保持されます。新しく始めたいときは /clear またはメニューの New Chat を使います。
  • 画面上の任意の場所を右クリックすると、カーソル位置にアプリケーションメニューが開きます。
  • 回答内のリンクはデフォルトブラウザで開きます。必要に応じてブラウザ側で続きを確認してください。
