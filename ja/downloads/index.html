初回起動の注意

macOSインストールのコツ

「開発元が未確認のため開けません」等の警告が出たら、アプリをControl+クリック→「開く」で許可。もしくは「設定 → プライバシーとセキュリティ」で許可します。

macOSのスクショ解析

スクリーンショット解析には「画面収録」の許可が必要です（設定 → プライバシーとセキュリティ → 画面収録）。

APIキーの準備

初回起動前に Google AI Studio で API キーを取得し、IrukaDark の設定で登録してください。

画面収録が許可されない場合

画面収録の許可がうまくいかないときは、ひとまず Option+S のショートカットからの解析開始を試すと成功しやすいです。
