瞬間AI起動

ショートカットでいつでもIrukaDarkを呼び出せます。

見えているもの全てを、AIが読み解く。

⌥Option + S

Option+S — スクショ解説

画像・グラフ・PDFなどを範囲選択してOption+S。画面上の内容をそのまま読み取って要約します。

⌥Option + ⇧Shift + S

Option+Shift+S — スクショ詳細解説

情報量の多いスライドや図解はOption+Shift+Sで詳細版を。細部まで丁寧に解説を受け取れます。

選択したテキストを、即座に解説。

⌥Option + A

Option+A — 概要解説

テキストを選択してOption+A。その場で要点を押さえた短い解説を表示します。

⌥Option + ⇧Shift + A

Option+Shift+A — 詳細解説

背景説明や手順が欲しいときはOption+Shift+A。より深い情報を含むロングアンサーを得られます。

⌥Option + ⌃Control + A

Option+Control+A — 図解

選択したテキストを図解画像にします。概念や関係性を視覚的に理解したいときに。

長文記事のインプットは、もう時間をかけない。

⌥Option + Q

Option+Q — URL概要

URLテキストを選択してOption+Q。ページを読み込んで要点だけを素早く解説します。

⌥Option + ⇧Shift + Q

Option+Shift+Q — URL詳細解説

Option+Shift+Qで選択したリンクを詳しく読み込み、重要セクションやポイントまで整理して届けます。

リライト×翻訳。想い通りのニュアンスを、あらゆる言語で。

⌥Option + E

Option+E — リライト

文章の意味を変えずに、より洗練された表現に書き換えます。

⌥Option + R

Option+R — 翻訳・言い換え

選択したテキストを自然な日本語・英語に翻訳、または噛み砕いて言い換え。Option+Rで即座に切り替えできます。

声だけで、AIとつながる。

Option+X — 音声入力

Option+Xを押して話すだけ。キーボードを使わずに、音声でAIに指示を出せます。

Option+Shift+Xで画面すべてを読み取り、音声入力した質問に回答します。

⌘Command + ⇧Shift + V

Command+Shift+V — クリップボードポップアップ

コピー履歴

IrukaDark は macOS のクリップボードと連携し、コピーしたテキストや画像を自動で記録します。テキストは最新1000件、画像は最新30件まで永続保存され、アプリを再起動しても履歴は残ります。過去にコピーした内容をすばやく呼び出して再利用できます。

⌥Option + ⇧Shift + V

Option+Shift+V — スニペットポップアップ

スニペット機能

よく使うフレーズやコード断片を登録しておくと、いつでも瞬時に引き出せます。繰り返し入力する定型文やコマンドを保存しておけば、タイピングの手間が大幅に減ります。

タイマー機能

集中作業やポモドーロテクニックに最適。設定した時間が来たら通知でお知らせします。

ストップウォッチ機能

作業時間の計測に便利。タスクにかかった時間を正確に把握できます。

スケジュール機能

指定した時間にアプリを起動させたり、Webサイトを開くことができます。オンライン会議URLをセットすれば、指定した時間に自動でアクセス。もう予定を忘れることはありません。つまり、「すみません。作業に集中していて遅れました。」という失態を犯すことはなくなったということです。集中タイムに没頭できます。「アイドルのLIVEチケットの販売開始時間が明日の12:00ぴったりだから、その時間になった瞬間にサイトを開くぞ」ということもできるということです。

@コマンドで、クリエイティブをその場で生成。

@image @image に続けてプロンプトを入力すると画像を生成できます。サイズや同時生成枚数は /image で設定します。
@video @video に続けてプロンプトを入力すると動画を生成できます。サイズ、秒数、画質、本数は /video で設定します。
@slide @slide に続けてプロンプトを入力するとスライド画像を生成できます。テンプレートや設定は /slide で管理します。

スラッシュコマンドで、かゆいところに手が届く。

解説を拡張

/what do you mean? 直前の解説の意味をもっと詳しく。背景や意図まで深掘りして補足してくれます。
/next さっきの解説の続きを依頼。再選択せずに話をどんどん掘り下げられます。
/table 直前の回答を表形式に整理。比較しやすく、ポイントが一目でわかります。
/translate 直前の解説を指定した言語に翻訳。対応言語は10種類以上で、言い換えにも使えます。

設定・モード切替

/web Web検索モードのオン/オフを切り替え。最新の検索結果を反映した解説にできます。
/image 画像生成の設定：サイズや同時生成枚数を調整できます。
/video 動画生成の設定：サイズ、秒数、画質、本数を調整できます。
/slide スライド生成の設定：テンプレート、枚数、スタイルを調整できます。

AI生成コマンド

自然言語で「やりたいこと」を伝えるだけで、IrukaDarkが適切なターミナルコマンドを生成。複雑な構文やフラグを覚える必要はありません。

直接実行機能

生成されたコマンドをIrukaDarkから直接実行できます。実行前にコマンドを確認でき、出力結果もアプリ内でリアルタイムに確認できます。

安全実行モード

IrukaDarkは破壊的な可能性のあるコマンドをハイライト表示し、明示的な確認を要求します。rm -rfのような危険な操作の誤実行を防ぎます。

    IrukaDark アプリ内の IrukaAutomation という別バイナリがアクセシビリティAPIを使用しています。macOSはバイナリ単位で権限を管理しており、アップデートでバイナリが置き換わるとコード署名のハッシュが変わり、権限が無効化されてアクセシビリティエラーが出ることがあります。

    解決方法1: 設定から再追加（推奨）

    1. システム設定 → プライバシーとセキュリティ → アクセシビリティ を開く
    2. IrukaDark を一度削除（−ボタン）
    3. 再度追加（＋ボタン）して /Applications/IrukaDark.app を選択
    4. アプリを再起動

    解決方法2: ターミナルでリセット

    # IrukaDarkのアクセシビリティ権限をリセット
tccutil reset Accessibility com.core.irukadark
# アプリを再起動後、権限ダイアログが表示されるので許可

    解決方法3: 完全リセット（上記で解決しない場合）

    # すべてのアクセシビリティ権限をリセット（他のアプリにも影響するので注意）
sudo tccutil reset Accessibility

    この問題はmacOSのセキュリティ仕様によるもので、署名付きアプリでもバイナリ更新時に発生することがあります。
