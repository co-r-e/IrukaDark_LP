Mac用ダウンロード
const result = await fetchUserData().then(data => {
return data.reduce((acc, user) => ({ ...acc, [user.id]: user }), {});
}).catch(error => console.error(error));
これはどういう意味？
この`reduce`メソッドは、配列の各要素（user）を取り出し、オブジェクトに変換しています。`acc`は累積器（accumulator）で、初期値は空のオブジェクト`{}`です。各ユーザーのIDをキー、ユーザーオブジェクトを値としたハッシュマップを作成しています。

お知らせ

    操作デモを見る

    テキスト選択や範囲キャプチャに反応して、その場でAIが解説します。

    パソコン作業が高速に捗る

    もう「わからない」で止まらない。瞬時理解で作業が加速します。

    瞬時解説ショートカット

    テキストや画面の気になる範囲を選んで専用ショートカットを押すだけ。状況に応じて最適なコマンドを切り替えながら、文脈に沿った解説がその場で得られます。

    フローを断ち切らない

    いちいち別のウィンドウに移動して、知りたいテキストをコピーして、AIの入力フォームに貼り付けて、回答を待つ、なんてことは不要です。その場でテキストを選択してOption+A、または範囲を選択してOption+Sを押すだけ。作業の流れを止めることなく、瞬時に疑問を解決できます。

    シンプルで安全

    IrukaDarkはデータベースを保持しません。あなたのデータはAIの学習に利用されず、プライバシーが守られます。

    使い方

    ショートカットでいつでもIrukaDarkを呼び出せます。

    Option+A — 即解説

    ⌥+A

    Mac: Option+A / Win・Linux: Alt+A

    テキストを選択してOption+A（Windows/LinuxはAlt+A）。その場で要点を押さえた短い解説を表示します。

    Option+Shift+A — 詳細解説

    ⌥+⇧+A

    Mac: Option+Shift+A / Win・Linux: Alt+Shift+A

    背景説明や手順が欲しいときはOption+Shift+A（Alt+Shift+A）。より深い情報を含むロングアンサーを得られます。

    Option+R — 翻訳・言い換え

    ⌥+R

    Mac: Option+R / Win・Linux: Alt+R

    選択したテキストを自然な日本語・英語に翻訳、または噛み砕いて言い換え。Option+R（Alt+R）で即座に切り替えできます。

    Option+1 — URL概要

    ⌥+1

    Mac: Option+1 / Win・Linux: Alt+1

    URLテキストを選択してOption+1（Alt+1）。ページを読み込んで要点だけを素早く解説します。

    Option+Shift+1 — URL詳細解説

    ⌥+⇧+1

    Mac: Option+Shift+1 / Win・Linux: Alt+Shift+1

    Option+Shift+1（Alt+Shift+1）で選択したリンクを詳しく読み込み、重要セクションやポイントまで整理して届けます。

    Option+S — スクショ解説

    ⌥+S

    Mac: Option+S / Win・Linux: Alt+S

    画像・グラフ・PDFなどを範囲選択してOption+S（Alt+S）。画面上の内容をそのまま読み取って要約します。

    Option+Shift+S — スクショ詳細解説

    ⌥+⇧+S

    Mac: Option+Shift+S / Win・Linux: Alt+Shift+S

    情報量の多いスライドや図解はOption+Shift+S（Alt+Shift+S）で詳細版を。細部まで丁寧に解説を受け取れます。

    使い方2

    スラッシュコマンドで直前の解説をさらに活用。

    /what do you mean?

    直前の解説の意味をもっと詳しく。背景や意図まで深掘りして補足してくれます。

    さっきの解説の続きを依頼。再選択せずに話をどんどん掘り下げられます。

    /web

    Web検索モードのオン/オフを切り替え。最新の検索結果を反映した解説にできます。

    よくある質問

    IrukaDarkに関するFAQ

    対応プラットフォームは？

    macOS に対応。ダウンロードページからインストーラを取得してそのまま使えます。

    翻訳ツールと何が違いますか？

    単語の置き換えではなく、文脈を踏まえた解説を提供します。意図や背景まで含めて、その場で自然な説明が得られます。

    データは保存されますか？

    保存しません。IrukaDarkはデータベースを保持せず、AI学習にも利用しないため、プライバシー保護に配慮しています。

    開発ツールは必要ですか？

    不要です。ダウンロードページからインストーラを入手し、案内に沿ってセットアップするだけで利用できます。

    素早く使うには？

    テキストを選択して Option+A。画像やグラフなどは範囲を選択して Option+S でスクリーンショット解説が利用できます。

    無料でオープンソースですか？

    はい。IrukaDark はオープンソースで無料です。ダウンロードページからいつでもインストールできます。

    インターネット接続は必要ですか？

    はい。解説を生成するためAIサービスに接続します。送信されるのは選択したテキストやキャプチャ範囲のみで、IrukaDark がデータを保存することはありません。

    OSごとの許可設定は？

    許可/設定 macOS
    画面キャプチャ 必須: 「画面収録」を許可
    アクセシビリティ 必須（ショートカットや画面上選択の操作に必要）
    管理者権限 不要
    ネットワーク（HTTPS） 送信を許可
    グローバルショートカット システム設定 → キーボード/ショートカット

    事前に必要なMacの設定は？

    項目 macOS
    開発ツール 一般利用では不要
    画面キャプチャ スクショ解析に「画面収録」の許可が必要
    インターネット HTTPS通信を許可
