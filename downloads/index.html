First‑run notes

macOS Install Tips

If you see a warning like “can’t be opened” or “from an unidentified developer”, Control‑click the app → Open, then confirm. Or allow it in System Settings → Privacy & Security.

macOS Screenshots

For screenshot analysis, grant Screen Recording when prompted (System Settings → Privacy & Security → Screen Recording).

Prepare Your API Key

Before the first launch, visit Google AI Studio to create an API key and paste it into IrukaDark’s settings.

Screen Recording Won’t Enable?

If Screen Recording permission keeps failing, try starting capture with the Option+S shortcut—it often succeeds on the first attempt.
